## Description

카페 쿠폰 시스템 
- 카페 사용자 등록
- 카페 사용자 쿠폰 적립, 조회, 사용
- 카페 관리자 쿠폰 관리

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
